Privacy Policy

This application is a Spotify statistics viewer that allows users to view their own listening data after authenticating with Spotify.

Data Access

The app only accesses Spotify data explicitly authorized by the user, such as top artists, top tracks, and recently played content.

Data Storage

No Spotify passwords are collected or stored. Access tokens are used only to fetch data from the Spotify Web API and are stored temporarily in the user's browser.

Data Sharing

No personal data is sold or shared with third parties.

User Control

Users can revoke access to this application at any time through their Spotify account settings.

Contact

If you have any questions, you can contact the developer via the project's GitHub repository.
